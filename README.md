## Run Locally

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd  letEndorse
```

Install dependencies

```bash
  npm install 
```

Start the server

```bash
  npm run start
```


## Deployment

To deploy this project run

```bash
  npm run deploy
```
